# สูตร: ค้นข้อมูล → สรุปเป็นรายงาน

ทำทีละขั้น อย่าข้าม:

1. **เข้าใจโจทย์** — ระบุให้ชัดว่า CEO อยากรู้อะไร ขอบเขตแค่ไหน
2. **ค้นหา** — `web_search` ด้วยคีย์เวิร์ดที่ตรงประเด็น (ลองหลายคำถ้าจำเป็น) → เลือก 3-5 แหล่งที่น่าเชื่อถือ
3. **อ่านจริง** — `fetch_url` ทีละแหล่ง จดประเด็นสำคัญ + ตัวเลข/ข้อเท็จจริง (อย่าเดา ถ้าไม่เจอให้บอกว่าไม่เจอ)
4. **เรียบเรียง** — จัดเป็นหัวข้อ: สรุปย่อ → ประเด็นหลัก → ข้อสังเกต → แหล่งอ้างอิง (ใส่ URL)
5. **บันทึก** — `write_file` เป็นไฟล์ `.md` ในโฟลเดอร์ที่เหมาะ (เช่น `research/<หัวข้อ>.md`)
6. **รายงาน CEO** — final สรุปสั้น 3-5 บรรทัด + ชี้ว่าไฟล์เต็มอยู่ไหน

กฎ: อ้างอิงเฉพาะสิ่งที่อ่านเจอจริง ห้ามแต่งข้อมูล ถ้าข้อมูลขัดแย้งกันให้บอกตามตรง
