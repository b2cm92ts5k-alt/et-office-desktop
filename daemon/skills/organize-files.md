# สูตร: จัดระเบียบไฟล์ใน workspace

ระวังสูงสุด — ไฟล์ของ CEO ห้ามหาย:

1. **สำรวจก่อน** — `list_dir` ทั่ว ๆ เข้าใจว่ามีอะไรบ้าง ไฟล์ประเภทไหน ซ้ำ/ขยะตรงไหน
2. **เสนอแผนจัด** — บอก CEO ว่าจะจัดยังไง (โครงโฟลเดอร์ใหม่ / ย้ายอะไรไปไหน / จะลบอะไร) **ก่อนลงมือ**
3. **สร้างโครง** — `mkdir` โฟลเดอร์ปลายทาง
4. **ย้าย** — `move` ทีละกลุ่ม (อย่าเขียนทับของที่มีอยู่ เช็คชื่อชนก่อน)
5. **ลบอย่างระวัง** — `delete` เฉพาะที่ยืนยันแล้วว่าเป็นขยะ/ซ้ำจริง ถ้าไม่แน่ใจให้ถามก่อน อย่าลบเดา
6. **สรุป** — final บอกว่าย้าย/ลบอะไรไปเท่าไร โครงใหม่เป็นยังไง

กฎเหล็ก: ทุก move/delete ผ่าน permission gate อยู่แล้ว — แต่ก่อนลบให้ดูเนื้อในก่อนเสมอ ถ้าไฟล์สำคัญหรือไม่แน่ใจ ห้ามลบ ให้รายงาน CEO แทน
